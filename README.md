# Air-Quality-Model
## Overview
This project predicts air quality levels using environmental sensor data such as CO, NO2, temperature, and humidity.

## Features
- Data Cleaning & Preprocessing  
- Feature Scaling & Selection  
- ML Models: Logistic Regression, KNN  
- Model Evaluation (Accuracy, Precision)  

## Dataset
- Real-world dataset 
- Includes pollution indicators and environmental features  

## Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib  
- Scikit-learn  

##  Results
- Successfully classified air quality levels  
- Improved accuracy using preprocessing techniques  

##  Future Improvements
- Add more models (Random Forest, XGBoost)  
- Deploy as web app
